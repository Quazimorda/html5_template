Бесплатный стартовый HTML5 шаблон для разработки сайтов с использованием Gulp. В него уже подключены библиотека JQuery и сетка Bootsrap. Для его работы вам понадобятся Git и Node Js. При желании вы можете также пользоваться Bower. Устанавливаемые с его помощью плагины попадут в папку app/libs. JQuery и сетка Bootsrap установлены именно так.

В папке app/sass расположены файлы стилей шаблона, которые собираются в последствии в app/css/main.css. Собственно стили пишутся с использованием препроцессора SASS. На этапе разработки я считаю удобным не минифицировать стили. Но в момент публикации сайта, советую раскоментировать соответствующие строчки в файле gulpfile.js в корневой папке шаблона.

Документация по проекту находится в разработке. Используйте, дорабатывайте, не стесняйтесь сообщать об ошибках в шаблоне.
